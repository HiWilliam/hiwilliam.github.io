mysql版本

问题原因

​ 经过实际中开发测试，laradock默认使用的时mysql最新版，.env文件中搜索MYSQL_VERSION

MYSQL_VERSION = latest可知，但是在使用时发现，最新的mysql对于账号验证有严格的要求，如果没有特殊版本要求，推荐是由5.6或者5.7

文件设置

​ 修改.env文件MYSQL_VERSION = 5.7，同样MySQL的相关账号配置也在.env下，可针对性进行设置操作
